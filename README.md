# MUICT UWU Pet Adoption
Ultimate Wilderness Utopia.

# Setup
1. Create `.env` in your local project's root.
2. Get your Google API Key (Mahidol email will not work).
3. Inside `.env`, add your API key to `GOOGLE_API_KEY=<YOUR_KEY>`.

# Running
1. `flutter run`
